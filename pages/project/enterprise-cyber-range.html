Enterprise Cyber Range

Dissertation Project September 2024

Overview

As part of my MSc. Cyber Security dissertation, I built a comprehensive red-blue team simulation lab designed to mimic a real-world enterprise network environment.

Technical Architecture

  • **Blue Team Stack:** Utilized **Wazuh SIEM** for real-time monitoring and event collection from endpoints running **Sysmon** and **Redline** for forensic data acquisition.
  • **Network:** Setup included Windows and Linux environments, **Active Directory** for authentication, and **pfSense** for firewall and network segmentation.
  • **Red Team Attacks:** Executed attacks using Kali Linux, **Metasploit**, Mimikatz, and nmap, meticulously mapping all techniques to the **MITRE ATT&CK framework**.

Conclusion

This project demonstrated end-to-end attack emulation, logging, forensic investigation, and defense readiness evaluation. It is a powerful tool for developing practical security skills and documenting detection gaps.
